Drop Inter and JetBrains Mono woff2 files here for fully offline typography.

Required (minimum):
  Inter-Regular.woff2
  Inter-Medium.woff2
  Inter-SemiBold.woff2
  Inter-Bold.woff2
  JetBrainsMono-Regular.woff2
  JetBrainsMono-Medium.woff2

Sources (download on an internet-connected host, then copy over):
  Inter        https://rsms.me/inter/
  JetBrains Mono  https://www.jetbrains.com/lp/mono/

If the files are missing the browser falls back to ui-sans-serif / ui-monospace.
